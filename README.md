# Sentiment Analysis of Amazon Product Reviews

This project performs sentiment analysis on Amazon product reviews using:
- Baseline models
- TF-IDF + Logistic Regression
- PyTorch Feed-Forward Neural Network

## Dataset
- Source: Kaggle
- Format: fastText
- Labels: positive / negative

## Models
- Majority baseline
- Rule-based keywords
- Logistic Regression (TF-IDF)
- PyTorch FFNN (TF-IDF)

## Results
Logistic Regression and PyTorch FFNN achieved similar performance with F1-score around 0.89.
